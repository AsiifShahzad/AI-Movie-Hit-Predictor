## Data File Naming Convention

Use this format for all data files:

```
[order]_[stage]_[description].csv

Examples:
01_raw_tmdb_10000_movies.csv     → First raw dataset
02_raw_tmdb_5000_movies.csv       → Second raw dataset
03_consolidated_movies.csv        → After consolidation
04_engineered_features.csv        → After feature engineering
05_models_predictions.csv         → Future (model predictions)
```
